Request = trình duyệt gửi yêu cầu.

Response = server trả dữ liệu.
# HTTP Notes

## HTTP

- Client gửi Request.
- Server trả Response.

## HTTP

Port: 80

## HTTPS

Port: 443

## GET

Dùng để lấy dữ liệu.

Ví dụ:
- Đọc báo
- Xem sản phẩm

## POST

Dùng để gửi dữ liệu.

Ví dụ:
- Đăng nhập
- Đăng ký
- Upload file

## Status Code

200 OK

301/302 Redirect

404 Not Found

500 Internal Server Error

## Header

Một số Header quan trọng:

- Host
- User-Agent
- Accept
- Cookie
- Content-Type

## Cookie

- Lưu trên Browser.
- Chứa Session ID.

## Session

- Lưu trên Server.
- Quản lý trạng thái đăng nhập.
